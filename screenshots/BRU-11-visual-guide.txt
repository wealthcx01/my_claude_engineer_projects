1. TIMER DISPLAY
   • Font Size: 4rem (64px)
   • Font Weight: 300 (Light)
   • Color: #ffffff (White)
   • Format: MM:SS with zero-padding
   • Initial Value: 25:00
   • Position: Centered within progress ring

2. PROGRESS RING
   • Type: SVG Circle
   • Outer Diameter: 300px
   • Radius: 140px
   • Stroke Width: 8px
   • Stroke Color: #ff6347 (Tomato Red)
   • Background Circle: #2a2a2a (Dark Gray, 20% opacity)
   • Animation: stroke-dashoffset transition (0.3s ease)
   • Circumference: 879.646px (2 × π × 140)
   • Starting Position: Top (rotated -90°)
   • End Cap: Round

3. COLOR THEME
   • Background: #1a1a1a (Dark)
   • Text: #ffffff (White)
   • Primary Accent: #ff6347 (Tomato Red)
   • Secondary: #2a2a2a (Dark Gray)

4. LAYOUT
   • Container: Flexbox (centered)
   • Alignment: center (both horizontal and vertical)
   • Gap: 3rem between timer and controls
   • Padding: 2rem

5. BUTTONS
   • Start/Pause:
     - Background: #ff6347
     - Padding: 1rem 2.5rem
     - Border Radius: 50px
     - Text: Uppercase, 1rem, weight 600
     - Hover: Lifts 2px with shadow

   • Reset:
     - Background: #2a2a2a
     - Same styling as Start/Pause

6. RESPONSIVE DESIGN (Mobile < 768px)
   • Timer Container: 250×250px (from 300×300)
   • Progress Ring Radius: 115px (from 140px)
   • Font Size: 3rem (from 4rem)
   • Buttons: Full width, stacked vertically
   • Max Button Width: 250px

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✓ Start/Pause Toggle
  - Starts countdown from current time
  - Updates every 1000ms (1 second)
  - Button text switches between "START" and "PAUSE"

✓ Reset Functionality
  - Returns timer to 25:00
  - Resets progress ring to full circle
  - Pauses timer if running

✓ Progress Ring Animation
  - Decreases smoothly as time counts down
  - Uses CSS transition (0.3s ease)
  - Calculated using: offset = circumference × (1 - progress)
  - Progress = timeRemaining / totalTime

✓ Timer Display Updates
  - Formats seconds as MM:SS
  - Zero-pads minutes and seconds
  - Updates in sync with progress ring
