Progress Ring Math:
  radius = 140px
  circumference = 2 × π × 140 = 879.646px

  Desktop: 300×300px container, r=140
  Mobile:  250×250px container, r=115 (circumference=722.566)

Timer Logic:
  Initial: timeRemaining = 1500 seconds (25 minutes)
  Interval: 1000ms (1 second)

  Each tick:
    timeRemaining--
    updateDisplay() → formats as MM:SS
    updateProgressRing() → calculates new offset

Progress Calculation:
  progress = timeRemaining / totalTime
  offset = circumference × (1 - progress)

  Example (at 24:57):
    progress = 1497 / 1500 = 0.998
    offset = 879.646 × (1 - 0.998) = 1.759px

Responsive Breakpoint:
  @media (max-width: 768px)
    - Timer size: 300→250px
    - Radius: 140→115px
    - Font: 4rem→3rem
    - Buttons: inline→stacked
